실습 프로젝트 Part 1: 기본 구조와 CSS 변수

학습 목표: CSS 변수, 기본 리셋, 레이아웃 시스템 이해

📚 Part 1 학습 내용

  • ✅ CSS 변수(Custom Properties) 활용법
  • ✅ 의료 시스템에 적합한 색상 팔레트 설계
  • ✅ 기본 리셋과 박스 모델 이해
  • ✅ 컨테이너 시스템과 반응형 기초
  • ✅ 타이포그래피 시스템 구축

🎯 실습 과제

  1. 1단계: CSS 변수를 수정해서 다른 색상 테마 만들기
  2. 2단계: --spacing 값들을 조정해서 간격 변화 확인
  3. 3단계: 새로운 CSS 변수 추가 (예: --color-accent)
  4. 4단계: 카드 컴포넌트에 호버 효과 추가

💡 핵심 포인트

  • CSS 변수: :root에서 정의하고 var()로 사용
  • 네이밍: --color-, --spacing-, --font- 등 접두사 활용
  • 유지보수: 한 곳에서 값 변경으로 전체 테마 변경 가능
  • 의료 테마: 신뢰감과 안정감을 주는 블루 계열 사용
