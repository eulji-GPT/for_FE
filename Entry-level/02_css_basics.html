1. CSS 선택자 (Selectors)

기본 선택자

/* 태그 선택자 */ p { color: blue; } /* 클래스 선택자 */ .highlight { background-color: yellow; } /* ID 선택자 */ #unique-id { color: red; font-weight: bold; } /* 전체 선택자 */ * { margin: 0; padding: 0; }

이것은 태그 선택자로 스타일이 적용된 단락입니다.

이것은 클래스 선택자로 하이라이트된 텍스트입니다.

이것은 ID 선택자로 스타일이 적용된 고유한 요소입니다.

3. 박스 모델 (Box Model)

모든 HTML 요소는 박스 모델을 가집니다: 내용(content) → 패딩(padding) → 테두리(border) → 마진(margin)

.box-example { width: 200px; height: 100px; padding: 20px; /* 내부 여백 */ margin: 20px; /* 외부 여백 */ border: 5px solid #3498db; /* 테두리 */ background-color: #ecf0f1; }

6. 위치 속성 (Position)

.position-relative { position: relative; /* 기준점 역할 */ } .position-absolute { position: absolute; /* 절대 위치 */ top: 20px; right: 20px; }

10. 반응형 디자인 (@media)

/* 화면 너비가 768px 이하일 때 */ @media (max-width: 768px) { .flex-container { flex-direction: column; /* 세로 배치 */ } .grid-container { grid-template-columns: 1fr; /* 1열로 변경 */ } }
