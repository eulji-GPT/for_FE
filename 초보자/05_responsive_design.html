1. 반응형 디자인이란?

반응형 웹 디자인(Responsive Web Design)은 다양한 화면 크기와 기기에서 최적의 사용자 경험을 제공하는 웹 디자인 접근법입니다.

/* 기본 메타 태그 - 반드시 포함! */ <meta name="viewport" content="width=device-width, initial-scale=1.0"> /* CSS 미디어 쿼리 */ @media (max-width: 768px) { /* 모바일 스타일 */ } @media (min-width: 769px) and (max-width: 1024px) { /* 태블릿 스타일 */ } @media (min-width: 1025px) { /* 데스크톱 스타일 */ }

📱 주요 브레이크포인트

  • 모바일: 320px ~ 767px
  • 태블릿: 768px ~ 1023px
  • 데스크톱: 1024px ~ 1199px
  • 대형 화면: 1200px 이상

2. 유연한 그리드 시스템

CSS Grid와 Flexbox를 활용한 반응형 그리드 시스템입니다.

/* 반응형 그리드 */ .responsive-grid { display: grid; gap: 20px; } /* 데스크톱 */ .grid-4 { grid-template-columns: repeat(4, 1fr); } /* 태블릿 */ @media (max-width: 1023px) { .grid-4 { grid-template-columns: repeat(2, 1fr); } } /* 모바일 */ @media (max-width: 767px) { .grid-4 { grid-template-columns: 1fr; } }

3. 반응형 카드 레이아웃

Auto-fit과 minmax를 사용한 완전 자동 반응형 카드 시스템입니다.

.card-container { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 20px; } /* 300px 미만으로는 줄어들지 않고, 공간이 있으면 자동으로 확장됩니다 */
🖼️ 반응형 이미지

Auto-fit 그리드

화면 크기에 따라 자동으로 열 개수가 조절됩니다. 최소 300px 너비를 유지합니다.

📱 모바일 최적화

모바일 친화적

작은 화면에서는 1열로, 큰 화면에서는 여러 열로 자동 배치됩니다.

💻 데스크톱 지원

유연한 확장

사용 가능한 공간을 효율적으로 활용하여 최적의 사용자 경험을 제공합니다.

4. 반응형 네비게이션

화면 크기에 따라 레이아웃이 변하는 네비게이션 바입니다.

.navbar-container { display: flex; justify-content: space-between; align-items: center; flex-wrap: wrap; } /* 모바일에서는 세로 배치 */ @media (max-width: 767px) { .navbar-container { flex-direction: column; text-align: center; } .navbar-menu { width: 100%; justify-content: center; } }

5. 반응형 폼

화면 크기에 따라 열 개수가 조절되는 스마트 폼입니다.

.form-grid { display: grid; gap: 20px; } /* 데스크톱: 3열 */ @media (min-width: 1200px) { .form-grid { grid-template-columns: repeat(3, 1fr); } } /* 태블릿: 2열 */ @media (min-width: 768px) and (max-width: 1199px) { .form-grid { grid-template-columns: repeat(2, 1fr); } } /* 모바일: 1열 */ @media (max-width: 767px) { .form-grid { grid-template-columns: 1fr; } }

6. 반응형 테이블

작은 화면에서는 가로 스크롤이 가능한 테이블입니다.

.table-responsive { overflow-x: auto; /* 가로 스크롤 */ } table { min-width: 600px; /* 최소 너비 설정 */ width: 100%; } /* 모바일에서 글자 크기 조절 */ @media (max-width: 767px) { th, td { padding: 10px; font-size: 0.9rem; } }

7. 고급 반응형 기법

CSS clamp() 함수

최소값, 선호값, 최대값을 설정하여 유연한 크기 조절이 가능합니다.

/* 폰트 크기 반응형 */ h1 { font-size: clamp(2rem, 5vw, 3rem); } /* 최소 2rem, 화면 크기의 5%, 최대 3rem */ /* 컨테이너 너비 */ .container { width: clamp(300px, 90vw, 1400px); margin: 0 auto; }

CSS 변수와 미디어 쿼리

:root { --font-size: 1rem; --padding: 20px; } @media (max-width: 767px) { :root { --font-size: 0.9rem; --padding: 15px; } } .element { font-size: var(--font-size); padding: var(--padding); }

Container Queries (최신 기법)

/* 컨테이너의 크기에 따른 스타일 */ .card-container { container-type: inline-size; } @container (min-width: 400px) { .card { display: flex; align-items: center; } }

화면 방향 대응

/* 세로 방향 */ @media (orientation: portrait) { .container { padding: 10px; } } /* 가로 방향 */ @media (orientation: landscape) { .navbar { position: fixed; top: 0; } }
