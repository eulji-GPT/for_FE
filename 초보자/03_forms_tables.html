1. 기본 폼 구조

HTML 폼은 사용자로부터 정보를 입력받기 위한 요소들의 집합입니다.

<form action="/submit" method="POST"> <!-- 폼 내용 --> <input type="submit" value="제출"> </form>

2. 다양한 입력 타입들

HTML5는 다양한 입력 타입을 제공합니다. 각 타입별로 브라우저가 자동으로 검증과 UI를 제공합니다.

<input type="text" placeholder="일반 텍스트"> <input type="email" placeholder="이메일 형식 검증"> <input type="password" placeholder="비밀번호 숨김"> <input type="number" min="0" max="100" placeholder="숫자만"> <input type="date" placeholder="날짜 선택"> <input type="time" placeholder="시간 선택"> <input type="color" placeholder="색상 선택"> <input type="range" min="0" max="100" placeholder="슬라이더"> <input type="file" accept="image/*" placeholder="파일 업로드"> <input type="url" placeholder="URL 형식 검증"> <input type="tel" placeholder="전화번호"> <input type="search" placeholder="검색어">

3. 테이블 (Table)

테이블은 구조화된 데이터를 표시하는 데 사용됩니다.

<table> <thead> <tr> <th>헤더1</th> <th>헤더2</th> <th>헤더3</th> </tr> </thead> <tbody> <tr> <td>데이터1</td> <td>데이터2</td> <td>데이터3</td> </tr> </tbody> </table>

4. 폼 검증 (Validation)

HTML5는 기본적인 폼 검증 기능을 제공합니다.

<!-- 필수 입력 --> <input type="text" required placeholder="필수 입력 필드"> <!-- 최소/최대 길이 --> <input type="text" minlength="4" maxlength="20" placeholder="4-20자"> <!-- 패턴 검증 --> <input type="text" pattern="[0-9]{3}-[0-9]{4}-[0-9]{4}" placeholder="010-1234-5678"> <!-- 숫자 범위 --> <input type="number" min="18" max="65" placeholder="18-65세">
